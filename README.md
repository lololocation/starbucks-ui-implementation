# 星巴克咖啡点餐小程序

这是一个仿星巴克咖啡的微信小程序点餐系统，使用原生微信小程序语法开发，实现了完整的咖啡店点餐功能。

## 功能特点

- **首页展示**：轮播图和推荐商品展示
- **菜单页面**：经典的左右联动布局，左侧分类导航，右侧商品列表
- **商品详情**：详细的商品信息展示，包括规格、配料、营养信息等
- **购物车功能**：添加商品到购物车，管理购物车内商品
- **订单管理**：查看历史订单和订单详情
- **个人中心**：用户信息和会员服务

## 技术特性

- 使用原生微信小程序语法（WXML, WXSS, JavaScript）
- 主色调采用星巴克绿色（#006241）
- 数据采用本地Mock数据，无需后端支持
- 实现了流畅的左右联动滚动效果

## 核心功能实现

### 菜单页面左右联动

菜单页面实现了经典的左右联动效果：

1. 左侧点击分类，右侧自动滚动到对应商品区域
2. 使用 `scroll-into-view` 属性实现精确滚动定位
3. 通过 `data` 中的 `toView` 字段控制滚动目标

### 数据结构

所有数据均使用本地Mock数据，包括：

- 商品分类信息
- 商品详细信息（名称、价格、描述等）
- 订单信息
- 用户信息

## 开发说明

此项目完全使用原生微信小程序语法开发，未使用任何第三方框架，便于理解和维护。所有图片资源均使用占位符代替，避免了外部依赖。

## 使用方法

1. 在微信开发者工具中打开此项目
2. 点击"编译"按钮运行项目
3. 测试各页面功能和交互效果

## 注意事项

- 此项目为演示用途，实际生产环境需接入真实后端接口
- 图片资源为占位符，实际使用时应替换为真实图片
- 样式可根据实际需求进行调整
